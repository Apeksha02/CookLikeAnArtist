About Cook Like An Artist


Cook Like An Artist is a website developed using HTML CSS JavaScript and The MealDB API Where we can search for a recipe by entering a single ingredient. After clicking on search various recipes having that ingredient will be visible with the name and image of that meal. And one option Get Recipe will be there after clicking on it we can see the step by step recipe and watch video option will be there by which we can see video for that recipe on YouTube!
